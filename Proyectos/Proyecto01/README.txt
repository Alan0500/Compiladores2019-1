                    Proyecto 01
        Analisis lexico del sub lenguaje de programacion Python p

Consiste en generar un analizador lexico del sub lenguaje de programacion Python (.p) y escribir las salidas del
analisis lexico (los archivos con los problemas propuestos en la clase) 


Para compilar el proyecto primero se debe posicionar en terminal donde esta el archivo pom.xml

Por seguridad primero limpiar cualquier archivo generado por maven

-- mvn clean

compilar el proyecto y ejecutar 

-- mvn compile exec:java -Dexec.mainClass="testmaven.Test"

Los archivos out son los generados por el ejemplo (se pueden borrar si se quiere observar el correcto funcionamiento de la clase)
